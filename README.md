A UIKit-based iOS application built using **Storyboards** that demonstrates
login & signup flow, navigation, and reusable UI components.

## Features
- UIKit + Storyboard architecture
- Login & Signup screens
- NavigationController-based flow
- Custom table view & collection view cells
- CocoaPods integration (Alamofire, RSLoadingView)
- Clean MVC-style separation

## Screens
- Login
- Signup
- Home
- Category listing
- Vertical & horizontal food collections

## Tech Stack
- Swift
- UIKit
- Storyboards
- CocoaPods
- Xcode
